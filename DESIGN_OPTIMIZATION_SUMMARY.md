# 設計優化總結

本次根據 `design-guide.md` 對整個應用進行了全面的設計優化。

## 優化內容

### 1. 動效系統（Motion System）

**更新位置：** `src/assets/base.css`

#### 1.1 時長 Tokens
- `--d-fast: 120ms` - hover 高亮、icon 變色、光標反饋
- `--d-base: 180ms` - 按鈕按下、輸入框聚焦、下拉展開
- `--d-slow: 240ms` - 卡片浮起、行 hover、輕量布局變化
- `--d-modal: 320ms` - 抽屜/彈窗進出、頁面切換

#### 1.2 緩動曲線 Tokens
- `--ease-std: cubic-bezier(0.2, 0.8, 0.2, 1)` - 標準動效
- `--ease-in: cubic-bezier(0, 0, 0.2, 1)` - 減速入場
- `--ease-out: cubic-bezier(0.4, 0, 1, 1)` - 加速離場

#### 1.3 位移與縮放 Tokens
- `--translate-light: 8px` - 內容淡入位移
- `--translate-drawer: 16px` - 抽屜位移
- `--scale-press: 0.98` - 按壓縮放

### 2. 色彩系統更新

**更新位置：** `src/assets/base.css`

- 主色調從藍色 (#0052d9) 改為綠色 (#00ad91)，更符合澳門生活品牌調性
- 優化背景色為 `#f7f8fa`，提供更好的視覺層次
- 更新邊框色為 `#e5e6eb`，更加克制和專業
- hover 背景色使用主色調的透明度變體 `rgba(0, 173, 145, 0.04)`

### 3. 陰影系統

遵循設計指南的三檔陰影系統：
- `--shadow-s1: 0 1px 3px rgba(0, 0, 0, 0.06)` - 輕微陰影
- `--shadow-s2: 0 6px 18px rgba(0, 0, 0, 0.08)` - 中等陰影
- `--shadow-s3: 0 12px 34px rgba(0, 0, 0, 0.12)` - 明顯陰影

### 4. Focus Ring（鍵盤用戶友好）

為所有可交互元素添加了專業的焦點樣式：
```css
outline: 2px solid rgba(0, 173, 145, 0.25);
outline-offset: 2px;
box-shadow: 0 0 0 1px var(--primary-color) inset;
```

### 5. 尊重"減少動效"設置

添加了對 `prefers-reduced-motion` 媒體查詢的支持，提升可訪問性。

## 性能與體驗

### 動效性能優化
- 優先使用 `transform` 和 `opacity` 實現動畫
- 避免觸發 layout 和 paint 的屬性變化
- 使用 CSS 變量統一管理，減少重複代碼

### 視覺層次
- 通過陰影、邊框、留白建立清晰的信息層級
- hover 狀態的微妙變化讓交互更加直觀
- 統一的間距系統保證視覺節奏

### 用戶體驗
- 所有交互都有及時的視覺反饋
- 動效短而輕，不會延緩用戶操作
- 移動端優化確保在小屏幕上的良好體驗

## 技術細節

### CSS 變量使用
所有動效、顏色、間距都通過 CSS 變量管理，便於：
- 統一調整和維護
- 支持主題切換（如需要）
- 減少硬編碼值

### 動畫性能
- 優先使用 `transform` 和 `opacity`
- 避免 `width`、`height`、`margin` 等觸發 reflow 的屬性
- 使用 `will-change` 提示瀏覽器優化（謹慎使用）

### 瀏覽器兼容性
- CSS 變量：支持所有現代瀏覽器
- cubic-bezier：廣泛支持
- prefers-reduced-motion：現代瀏覽器支持

## 總結

本次優化全面遵循了 `design-guide.md` 的設計原則，重點在：

1. **建立系統化的動效語言**：統一的時長、曲線、位移參數
2. **提升交互品質**：完整的狀態、及時的反饋、流暢的動效
3. **優化視覺層次**：克制的陰影、合理的留白、清晰的結構
4. **增強可訪問性**：鍵盤友好、減少動效支持、合適的點擊目標
5. **精細化響應式**：多斷點適配、移動端優化

所有改動都經過深思熟慮，既保證了視覺品質，又確保了性能和可維護性。
